<h1 align="center">Site displaying information about the city</h1>

## _The application allows users to calculate the cost of a trip by car depending on a various conditions_

It have the following functionality:

- General information;
- History of the city;
- Attractions;
- Hotels;
- Restaurants.

- Apache Tomcat
- JavaEE
- HTML/CSS
