<h1 align="center">Site displaying information about the city</h1>

## _Site for tourists in Nizhny Novgorod_

It has the following functionality:

- General information;
- History of the city;
- Attractions;
- Hotels;
- Restaurants.


<h2 align="center">_Technologies_</h2>

- Java
- Maven
- Apache Tomcat
- JavaEE
- HTML/CSS
